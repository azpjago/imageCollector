# Made with love, by azpjago aka Asep saepul Anwar

This python program can collect image data, whatever you want for training machine learning.
The image is from google photos, and using serpAPI key.
You can make it easy to get serpAPI key, from https://serpapi.com this is free API for 300 query in a month.
To use this program, you can clone it:
1. open your terminal, and access the project folder, i use linux mint btw
2. git clone https://github.com/azpjago/imageCollector.git
3. then, pip install -r requirements.txt
4. wait to finish the install process
5. then, you can run the collect_data_gui.py write the "python collect_data_gui.py" in the terminal section
6. then, you can choose what image you want to collect, and other atribut from the image, example if you want to find banana image, we know that banana has a lot type in ripeness, then you can pick unripe, ripe, or overripe
7. you can also pick how much many image you want to download in category

The image, is for training machine learning, image classification based.
